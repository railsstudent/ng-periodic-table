# ng-periodic-table

Build a dynamic period table similar to ptable.com

## Deploy to github.io

* Method 1:
  * ng build --prod --base-href "https://railsstudent.github.io/ng-periodic-table/"
  * ngh --no-silent
* Method 2:
  * npm run deploy
